# Lichtenberg - 世界文学日报

📖 每日自动抓取美洲、欧洲、非洲、亚洲四大洲文学期刊与评论

## 关于

Lichtenberg 是一个世界文学情报追踪系统，自动聚合全球重要文学媒体的最新内容。

## 覆盖区域

- 🌎 **美洲**: 美国、加拿大、拉丁美洲
- 🌍 **欧洲**: 英国、法国、德国、意大利、西班牙、俄罗斯、北欧
- 🌍 **非洲**: 尼日利亚、南非及泛非文学
- 🌏 **亚洲**: 中国、日本、韩国、印度、东南亚、中东
